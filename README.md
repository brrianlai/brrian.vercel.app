# [brrian.vercel.app](https://brrian.vercel.app)

Built with React and Next.js

### Cloning
```
git clone https://github.com/brrianlai/brrian.vercel.app
cd brrian.vercel.app

npm install
```

### Starting

```
npm run dev
```
Starts a local server on port 3000: `localhost:3000`

### Building

```
npm run build
```
